# druidfi/docker-s3cmd

Container to sync backups to Amazon S3 and get them from there.

## Includes

- Alpine image
- s3cmd 2.0.2

## s3cmd documentation

- See the documentation in https://s3tools.org/usage
- Sync specific documentation in https://s3tools.org/s3cmd-sync
